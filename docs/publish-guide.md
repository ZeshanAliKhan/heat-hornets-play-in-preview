# Publish Guide

## Expected repository name

`heat-hornets-play-in-preview`

## Expected GitHub Pages URL

`https://zeshanalikhan.github.io/heat-hornets-play-in-preview/`

## Steps

1. Create a public GitHub repository named `heat-hornets-play-in-preview`.
2. Push this project folder to `main`.
3. Open repository `Settings`.
4. Open `Pages`.
5. Set:
   - `Deploy from a branch`
   - `main`
   - `/root`
6. Save.
